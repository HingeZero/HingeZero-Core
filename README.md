HingeZero
HingeZero is a deterministic associative memory mechanism built around a hinge-at-zero nonlinearity.
The framework treats the neighbourhood around zero as an active equilibrium region rather than a passive dead zone. Opposing states can balance, hold structure, and resolve instead of collapsing immediately.
Core implementation is intentionally minimal (~9 executable lines).
Features:
• Deterministic updates
• Noise tolerant recall dynamics
• Quantisation friendly
• Minimal implementation
• Reproducible
• Easy integration
Repository contains only the HingeZero core.
No tests.
No wrappers.
No hybrids.
Just HingeZero.
